# 데모 랩 카탈로그 정책

이 카탈로그는 해커톤 심사 흐름을 재현하기 위한 결정적 데모 픽스처다. 실제 교수, 실제 연구실 목록, 모집 또는 채용 상태를 표현하지 않는다.

## 데이터 의미

- 모든 교수명과 연구실명은 가상 데모 값이다.
- `dataStatus`는 항상 `demo`이며, 서비스는 이를 실제 운영 데이터처럼 표시하면 안 된다.
- `labUrl`은 해당 대학의 컴퓨터공학 계열 공식 학과 랜딩 페이지다.
- `officialSourceUrl`은 해당 대학 또는 학과의 공식 홈페이지다.
- `verifiedAt`은 링크 접근 경로를 확인한 시각이다. 교수 소속, 연구 주제, 모집 여부 또는 정보 최신성을 검증한 시각이 아니다.

## 출처와 한계

공식 링크는 기관·학과 도메인으로 이동할 수 있다는 근거만 제공한다. 가상 교수명과 연구실명에 대한 근거가 아니며, 국내 대학원 연구실의 완전한 목록이나 실시간 크롤링 결과라고 주장하지 않는다. 공식 링크가 존재하더라도 특정 연구실이 학생을 모집한다는 의미는 아니다.

심사 데모에서는 데이터가 가상임을 밝히고 공식 출처 링크와 확인 시각을 함께 노출해야 한다.

## 운영 데이터 전환 조건

실데이터로 전환하려면 레코드별 실제 연구실 페이지, 교수 공식 프로필, 수집 시각, 갱신 주기와 삭제 요청 절차를 별도 파이프라인에서 확보해야 한다. 자동 수집을 도입하기 전 각 사이트의 이용약관과 `robots.txt`를 확인하고, 허용된 범위와 요청 속도를 준수해야 한다. 이 조건을 충족하기 전에는 데모 레코드를 실제 교수 추천으로 재사용하지 않는다.
